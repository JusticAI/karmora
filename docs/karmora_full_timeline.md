## 2026-04-18 — 技术调研

### 对话内容

**用户：** "我获取不到reddit的开发者client ID和secret，你在做技术调研的时候需要注意这个点"

**用户：** "这是申请API的导读，你去看看"（附链接）

### 解决了什么
- 关键约束：没有 Reddit API 凭证（Client ID + Secret）
- 调研发现：Chrome 扩展可以借用用户已登录身份进行同源请求
- Reddit 封的是"外部程序直接调API"，但从浏览器内部发请求不受影响

### 产生了什么
- **关键技术决策：Chrome 扩展 + PullPush API 方案**（绕过 Reddit API 限制）
- 区分了 Reddit 三种开发者工具：
  - Devvit：在Reddit里跑的小程序/游戏
  - Data API：从外部读数据（需要凭证）
  - Embeds：嵌入Reddit内容到外部网站

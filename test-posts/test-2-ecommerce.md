## 预期检测结果

- 🔴 **高风险 — 多项违规**
- **违规 1：直接推广链接** — r/ecommerce Rule 6: No self-promotion
  - 含产品推广链接（producthunterpro.com/?ref=karma123）
  - 含博客引流链接（myecomjourney.com）
- **违规 2：索赞（Upvote solicitation）** — Reddit 全局规则
  - "please upvote this so more people can see it"
- **违规 3：联系方式引流** — 社区规则禁止公开留联系方式
  - 公开留了邮箱地址
- **违规 4：标签滥用** — r/ecommerce 不支持 hashtag
- **后果：** 帖子被删，账号可能被标记
- **修复建议：** 删除所有外链和推广内容，纯分享经验，不提具体工具名

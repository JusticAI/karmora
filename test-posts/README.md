# Karmora 规则检查器 — 测试帖子

> 三篇测试帖子，用于验证 Karmora 的规则检查功能是否能正确检测违规风险。

## 测试场景

| # | 社区 | 测试违规类型 | 预期检测结果 |
|---|------|-------------|-------------|
| 1 | r/food | 标题格式违规（缺少 [tag]） | 高风险：标题无标签会被删帖 |
| 2 | r/ecommerce | 自我推广 + 链接违规 | 高风险：含推广链接会被删帖 |
| 3 | r/gaming | 索赞 + 低质量内容 | 中风险：索赞会被踩，内容空洞 |

## 使用方法

1. 打开 Karmora 规则检查器：https://reddit-karma-landing.vercel.app/assistant.html
2. 选择对应社区
3. 粘贴帖子标题和内容
4. 查看风险评分和违规项

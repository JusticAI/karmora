## 预期检测结果

- 🟠 **中风险 — 多项轻度违规**
- **违规 1：索赞（Upvote solicitation）** — Reddit 礼仪 Reddiquette
  - "Upvote if you think this is..."
  - "⬆️⬆️⬆️" 箭头暗示
- **违规 2：低信息量内容** — r/gaming Rule 1: Submissions must be directly gaming-related
  - 帖子主要是"炫耀捡漏"，缺乏游戏相关内容讨论
- **违规 3：自我推广引流** — YouTube 频道链接
  - EDIT 2 中引流到个人 YouTube 频道
- **违规 4：标题质量差** — 标题含过多 emoji 和感叹号
- **后果：** 可能被踩到 0 分，被 mod 标记
- **修复建议：** 去掉索赞语句，补充游戏体验内容，删除 YouTube 引流

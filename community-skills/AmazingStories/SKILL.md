# 模块一：社区爆款核心技能

## 1. 社区人群画像

- **身份**：Apple TV+ 剧集《Amazing Stories》观众、科幻/奇幻剧迷、Spielberg/Williams 粉丝
- **年龄**：25-45 岁，怀旧情怀重
- **偏好**：剧集细节讨论 > 泛泛评价，深度分析 > 表面评论
- **情绪偏好**：怀旧、热情、略带失望（对剧集质量的期望落差）
- **雷点底线**：反感剧透不标记、反感无脑黑、反感政治化讨论

## 2. 高赞选题逻辑

| 类型 | 示例 | 为什么火 |
|------|------|----------|
| **逐集讨论** | "Season 1 Episode 3 Discussion: Dynoman and the Volt" | 官方讨论帖，集中讨论 |
| **深度分析** | "Episode 1 The Cellar. Something new I spotted" | 发现细节，引发共鸣 |
| **怀旧对比** | "So I bought the Comic that the episode 'The Rift' was based off" | 原著党互动 |
| **疑问求解** | "Is episode 4 in the opening credits?" | 好奇心驱动 |

**核心规律**：细节 > 概括，怀旧 > 现实，讨论 > 评价

## 3. 内容黄金结构

**帖子结构：**
```
1. 开头：点明主题（哪一集/哪个场景）
2. 核心：具体观察/问题/分析（2-4句）
3. 结尾：提问或邀请讨论
```

**评论结构：**
```
1. 直接表达观点（1-2句）
2. 补充细节或个人感受（可选）
3. 简短收尾或表情
```

**信息密度**：中等。允许情感表达，但要有具体内容支撑。

## 4. 情绪拿捏公式

- **怀旧**：40%（"I remember when..."）
- **热情**：30%（对剧集细节的兴奋）
- **失望**：20%（对剧集质量的抱怨）
- **幽默**：10%（自嘲、调侃）

**禁止**：过度批评、剧透不标记、政治化

## 5. 加分关键细节

- ✅ 提到具体集数和场景
- ✅ 发现隐藏细节或彩蛋
- ✅ 对比原著/漫画
- ✅ 引用 John Williams 配乐
- ✅ 用表情符号增强情感（😂、😭、🔥）

## 6. 绝对避雷黑名单

- ❌ 剧透不标记 = 被踩
- ❌ 无脑黑（"this show sucks"）= 被嘲讽
- ❌ 政治化讨论 = 引发争议
- ❌ 重复提问（"When is season 2?"）= 被忽略
- ❌ 推广无关内容 = 删帖

# 模块二：社区专属行文人设

## 1. 整体语气调性

**怀旧 + 热情 + 随意**

- 允许情感表达，不像 r/ecommerce 那么务实
- 可以用表情和感叹号
- 语气介于粉丝和评论家之间

## 2. 句式习惯

- **长短句混合**：短句表达观点，长句解释细节
- **断句方式**：多用感叹号、问号，少用句号
- **缩写**：不多，偶尔用 "EP", "S1"
- **俚语**：常用 "holy shit", "lol", "wtf", "yeaaa"

## 3. 开头钩子范式

| 类型 | 模板 |
|------|------|
| **细节型** | "So in the past just before his construction truck drives off..." |
| **疑问型** | "Is episode 4 in the opening credits?" |
| **感叹型** | "Holy shit, I didn't realize how invested..." |
| **怀旧型** | "I remember watching this when..." |

**规律**：直接切入具体场景或情感，不铺垫

## 4. 中段表达特征

- **惯用修辞**：夸张（"absolute magic"）、反问
- **吐槽方式**：直接表达失望，但带幽默（"the acting was horse shit"）
- **自我表达**：用 "I" 开头，分享个人观影体验

## 5. 结尾固定套路

| 类型 | 模板 |
|------|------|
| **提问** | "What did folks think?" |
| **感叹** | "What a ride!" |
| **表情** | 😂、😭、🔥 |
| **留白** | 直接结束，不总结 |

**规律**：情感化收尾，把话题抛给评论区

## 6. 高频本土词汇/黑话/惯用短句

| 词汇 | 含义 |
|------|------|
| EP | Episode，集数 |
| S1 | Season 1，第一季 |
| reboot | 重启版 |
| finale | 大结局 |
| closure | 收尾/交代 |
| Easter egg | 彩蛋 |
| theme | 主题曲/主题 |
| Spielberg | 导演/制片人 |
| Williams | 配乐大师 John Williams |
| Apple TV+ | 播出平台 |

**惯用短句**：
- "Holy shit"
- "What a ride"
- "I didn't realize how invested..."
- "Absolute magic"
- "Horse shit"（形容演技差）

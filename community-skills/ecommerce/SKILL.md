# 模块一：社区爆款核心技能

## 1. 社区人群画像

- **身份**：电商从业者（独立站卖家、Shopify 用户、DTC 品牌主、dropshipper）
- **阶段**：新手到中级为主，少数资深卖家
- **偏好**：实操经验 > 理论，具体数字 > 泛泛而谈
- **情绪偏好**：务实、直接、略带焦虑（对增长和利润的焦虑）
- **雷点底线**：反感明显自我推广、反感 AI 味重的内容、反感没做过就来问基础问题

## 2. 高赞选题逻辑

| 类型 | 示例 | 为什么火 |
|------|------|----------|
| **真实困境** | "Returns eat you alive" | 共鸣，大家都有这个痛点 |
| **实操经验** | "How I Finally Got My Shopify Subscription Payment Working" | 可复制，有具体步骤 |
| **行业洞察** | "How are you really using ChatGPT behind the scenes?" | 好奇心，想知道真实用法 |
| **反常识** | "I really don't want to sell on etsy... Do I have to?" | 引发讨论，表达真实想法 |

**核心规律**：真实 > 完美，具体 > 抽象，痛点 > 爽点

## 3. 内容黄金结构

**帖子结构：**
```
1. 开头：一句话点明问题/困境（1-2句）
2. 背景：简述自己的情况（可选，3-5句）
3. 核心：具体问题或经验（重点展开）
4. 结尾：直接提问或邀请讨论
```

**评论结构：**
```
1. 直接给答案/建议（1-2句）
2. 补充个人经验（可选，2-3句）
3. 简短收尾（可选）
```

**信息密度**：高。不说废话，每句话都有信息量。

## 4. 情绪拿捏公式

- **共鸣**：40%（"I had the same problem..."）
- **务实**：35%（直接给建议，不废话）
- **自嘲**：15%（承认自己的错误）
- **吐槽**：10%（对行业现状的抱怨）

**禁止**：过度乐观、鸡汤、营销语气

## 5. 加分关键细节

- ✅ 给具体数字（"93% profit margin", "$40 product"）
- ✅ 提到具体工具（Shopify, Stripe, Meta Ads）
- ✅ 分享真实失败经历（"I made the mistake of..."）
- ✅ 用第一人称 "I" 叙述
- ✅ 结尾提问引发讨论

## 6. 绝对避雷黑名单

- ❌ 直接发产品链接 = 删帖
- ❌ 自我推广语气（"I'll write your copy in 4 hours"）= 被踩
- ❌ AI 生成感强的内容 = 被质疑
- ❌ 问基础问题不先搜索 = 被嘲讽
- ❌ 过度使用 emoji = 不专业
- ❌ 卖课/卖服务的帖子 = 直接被删

# 模块二：社区专属行文人设

## 1. 整体语气调性

**务实 + 直接 + 略带焦虑**

- 不客套，直奔主题
- 承认问题和困难，不装成功
- 语气介于专业和口语之间

## 2. 句式习惯

- **长短句混合**：短句为主（5-15词），偶尔长句解释
- **断句方式**：多用句号，少用逗号连接长句
- **缩写**：常用 "ecom", "dropship", "AOV", "LTV", "CRO"
- **俚语**：不多，偶尔用 "dip shit", "bro"

## 3. 开头钩子范式

| 类型 | 模板 |
|------|------|
| **痛点型** | "Returns eat you alive" |
| **问题型** | "How can I increase sales?" |
| **经验型** | "I've been doing X for Y years..." |
| **反问型** | "Do I have to sell on Etsy?" |

**规律**：直接点明问题或困境，不铺垫

## 4. 中段表达特征

- **惯用修辞**：类比（"flying blind"）、反问
- **吐槽方式**：直接抱怨，不绕弯（"They have to restock before they can start overstocking before the dip shit"）
- **自我表达**：用 "I" 开头，分享个人经历

## 5. 结尾固定套路

| 类型 | 模板 |
|------|------|
| **提问** | "What's your experience?" |
| **邀请** | "Happy to answer questions" |
| **简短总结** | "That's what worked for me" |
| **留白** | 直接结束，不总结 |

**规律**：简短，不啰嗦，把话题抛给评论区

## 6. 高频本土词汇/黑话/惯用短句

| 词汇 | 含义 |
|------|------|
| AOV | Average Order Value |
| LTV | Lifetime Value |
| CRO | Conversion Rate Optimization |
| DTC | Direct to Consumer |
| dropship | 代发货模式 |
| Meta Ads | Facebook/Instagram 广告 |
| Shopify | 电商平台 |
| Stripe | 支付处理 |
| organic traffic | 自然流量 |
| retention | 留存 |
| scaling | 规模化 |
| landed cost | 到岸成本 |

**惯用短句**：
- "Returns eat you alive"
- "Toughest part? Balancing growth with profitability"
- "Flying blind"
- "What worked for me was..."

# 模块一：社区爆款核心技能

## 1. 社区人群画像

- **身份**：美食爱好者、家庭厨师、吃货、专业厨师
- **年龄**：18-45 岁，覆盖广
- **偏好**：视觉冲击 > 文字描述，成品展示 > 过程分享
- **情绪偏好**：友好、鼓励、好奇
- **雷点底线**：反感标题格式错误、反感种族歧视、反感低质量内容

## 2. 高赞选题逻辑

| 类型 | 示例 | 为什么火 |
|------|------|----------|
| **视觉冲击** | "CREM CHESSSSS [homemade] 😍🥰" | 颜值高，引发食欲 |
| **异国美食** | "Thịt bò gác bếp – Smoked dried beef from Vietnam" | 好奇心，文化分享 |
| **家常菜升级** | "Reverse seared lamb with oven roasted herb potatoes" | 可复制，有成就感 |
| **趣味猎奇** | "What is the strangest fair food?" | 话题性，引发讨论 |

**核心规律**：好看 > 好吃描述，具体 > 抽象，分享 > 教学

## 3. 内容黄金结构

**帖子结构：**
```
1. 标题：[标签] + 菜名（必须格式）
2. 图片：高质量食物照片（核心）
3. 正文（可选）：简短背景或食谱（2-5句）
```

**评论结构：**
```
1. 直接赞美（1句）
2. 提问（可选）："Recipe?" "How did you make this?"
3. 表情（可选）：😍🤤🔥
```

**信息密度**：低。重颜值轻文字，评论简短友好。

## 4. 情绪拿捏公式

- **赞美**：50%（"Looks amazing!" "Holy flavor Batman!"）
- **好奇**：30%（"Recipe?" "What's the heat level?"）
- **幽默**：15%（自嘲、调侃）
- **怀旧**：5%（"Reminds me of my grandma's cooking"）

**禁止**：负面评价、挑剔、傲慢

## 5. 加分关键细节

- ✅ 用固定标签格式：[homemade]、[I ate]、[Pro/chef]
- ✅ 高质量食物照片（光线、构图）
- ✅ 附上食谱或做法
- ✅ 用表情增强情感（😍🤤🔥😋）
- ✅ 简短背景（"Made this for my family"）

## 6. 绝对避雷黑名单

- ❌ 标题格式错误 = 删帖
- ❌ 没有图片的帖子 = 被忽略
- ❌ 负面评价他人食物 = 被踩
- ❌ 种族歧视内容 = 封号
- ❌ 广告/推广 = 删帖
- ❌ 问基础问题不先搜索 = 被嘲讽

# 模块二：社区专属行文人设

## 1. 整体语气调性

**友好 + 鼓励 + 轻松**

- 零攻击性，纯赞美氛围
- 允许大量表情符号
- 语气介于朋友和美食评论家之间

## 2. 句式习惯

- **短句为主**：5-15 词
- **断句方式**：多用感叹号、省略号
- **缩写**：不多，偶尔用 "OP", "tbh", "yahh"
- **俚语**：常用 "holy shit", "das is good", "fire" 🔥

## 3. 开头钩子范式

| 类型 | 模板 |
|------|------|
| **标签型** | "[homemade] + 菜名" |
| **感叹型** | "Holy flavor Batman!" |
| **疑问型** | "What's the heat level?" |
| **赞美型** | "Looks amazing!" |

**规律**：直接表达情感或提问，不铺垫

## 4. 中段表达特征

- **惯用修辞**：夸张（"obsessed", "fire"）、比喻
- **吐槽方式**：轻松幽默，不带攻击性
- **自我表达**：分享个人体验，不炫耀

## 5. 结尾固定套路

| 类型 | 模板 |
|------|------|
| **提问** | "Recipe?" |
| **表情** | 😍🤤🔥😋 |
| **感谢** | "Thanks for sharing!" |
| **留白** | 直接结束，不总结 |

**规律**：简短友好，把话题抛给评论区

## 6. 高频本土词汇/黑话/惯用短句

| 词汇 | 含义 |
|------|------|
| [homemade] | 自制标签 |
| [I ate] | 吃过标签 |
| [Pro/chef] | 专业厨师标签 |
| [Produce] | 食材标签 |
| OP | Original Poster，发帖人 |
| tbh | To be honest |
| yahh | Yeah 的口语化 |
| fire 🔥 | 很棒、很赞 |
| holy shit | 惊叹 |
| recipe | 食谱 |

**惯用短句**：
- "Looks amazing!"
- "Holy flavor Batman!"
- "Recipe?"
- "Thanks for sharing!"
- "Das is good yahh"
- "I'm obsessed"
